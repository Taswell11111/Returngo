APIs for Return Management



GET
/rmas
List and search RMAs


Use this API to get all RMAs within a search criteria.

Use the cursor argument to get the next batch.

Parameters
Cancel
Name	Description
pagesize *
number
(query)
The number of RMAs to include in the response. Valid values are 1 through 500.

1
cursor
string
(query)
The position to start from, as received from a previous call. If not present, start from the beginning

cursor
sort_by
string
(query)
The column names to sort by.

Each column name should be preceded by - for descending and + for ascending.

sort_by can be a comma separated list, for example: ?sort_by=+rma_created_at,-rma_updated_at.

NOTE: the + sign must be url encoded to %2B

sort_by
rma_created_at
string
(query)
Filter by criteria on the created_at column

Allowed criteria: lt,lte,gt,gte,eq,neq

Example: ?rma_created_at=lte:2021-01-20T00:00:00

The date value should be in ISO 8601 format YYYY-MM-DDTHH:mm:ss

rma_created_at
rma_updated_at
string
(query)
Filter by criteria on the updated_at column

Allowed criteria: lt,lte,gt,gte,eq,neq

Example: ?rma_updated_at=lte:2021-01-20T00:00:00

The date value should be in ISO 8601 format YYYY-MM-DDTHH:mm:ss

rma_updated_at
status
string
(query)
A comma delimited status list, the accepted values are:

Pending

Approved

Received

Validated

Done

Rejected

Canceled

Pending
order_name
string
(query)
A comma delimited list of order names e.g order_name=1001,1002 ...

order_name
customer_ids
string
(query)
A comma delimited list of customer ids e.g customer_ids=6898673516706,6433510031522 ...

customer_ids
tracking_numbers
string
(query)
A comma delimited list of tracking numbers e.g tracking_numbers=113498651924,390730297459 ...

tracking_numbers
x-shop-name *
string
(header)
The shop to operate on, e.g. example.myshopify.com.

To find your shop URL, click on the "copy" icon next to your store name at the top of the ReturnGO dashboard.

This will copy your shop URL (which is not visible) to your clipboard.
